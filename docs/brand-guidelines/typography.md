# Eagles Healing Nest Connect - Typography

## Font Stack

### Primary Font Family
```css
font-family: 'Segoe UI', Tahoma, Geneva, Verdana, sans-serif;
```

**Rationale**: Segoe UI is modern, clean, and highly accessible. The fallback chain ensures consistent rendering across platforms.

### Monospace Font (Code)
```css
font-family: 'Courier New', Courier, monospace;
```

## Font Sizes

### Headings
- **H1**: 2.5rem (40px) - Main page titles
- **H2**: 2rem (32px) - Section headings
- **H3**: 1.5rem (24px) - Subsection headings
- **H4**: 1.25rem (20px) - Minor headings
- **H5**: 1.1rem (18px) - Small headings
- **H6**: 1rem (16px) - Smallest headings

### Body Text
- **Regular**: 1rem (16px)
- **Small**: 0.9rem (14px)
- **Large**: 1.1rem (18px)

### Mobile Adjustments
- **H1 Mobile**: 1.75rem (28px)
- **H2 Mobile**: 1.35rem (22px)
- **H3 Mobile**: 1.1rem (18px)

## Font Weights

- **Regular**: 400 - Body text, paragraphs
- **Medium**: 500 - Emphasis
- **Semi-Bold**: 600 - Labels, headings
- **Bold**: 700 - Strong emphasis

## Line Heights

- **Headings**: 1.2
- **Body Text**: 1.6
- **Form Labels**: 1.4

## Spacing

### Paragraph Margins
- **Bottom margin**: 1em
- **Top margin**: 0

### Heading Margins
- **Bottom margin**: 0.5em
- **Top margin**: 0 (or 1.5em for section headings)

## Accessibility Notes

1. **Minimum font size**: 16px for body text
2. **Sufficient line height**: 1.5x for improved readability
3. **Adequate spacing**: Between paragraphs and elements
4. **High contrast**: Text must meet WCAG AA standards (4.5:1)
5. **Avoid**: All-caps text for extended reading
6. **Letter-spacing**: Normal or slightly increased for readability

## Usage Guidelines

1. Use **H1** only once per page for main title
2. Use heading hierarchy correctly (don't skip levels)
3. Ensure **sufficient contrast** between text and background
4. Use **bold** sparingly for emphasis
5. Keep line length between **50-75 characters** for optimal readability
6. Use **ems** for responsive typography
