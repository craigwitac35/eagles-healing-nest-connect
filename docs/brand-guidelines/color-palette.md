# Eagles Healing Nest Connect - Color Palette

## Primary Colors

### Primary Blue
- **Hex**: #2c5aa0
- **RGB**: rgb(44, 90, 160)
- **HSL**: hsl(213, 57%, 40%)
- **Usage**: Headers, navigation, primary buttons, links
- **Purpose**: Trust, stability, professionalism

### Secondary Blue
- **Hex**: #6db3f2
- **RGB**: rgb(109, 179, 242)
- **HSL**: hsl(213, 85%, 69%)
- **Usage**: Hover states, accents, secondary buttons
- **Purpose**: Approachability, friendliness

### Accent Orange
- **Hex**: #f39c12
- **RGB**: rgb(243, 156, 18)
- **HSL**: hsl(38, 89%, 51%)
- **Usage**: Call-to-action elements, highlights, focus states
- **Purpose**: Energy, warmth, attention

## Neutral Colors

### Dark Text
- **Hex**: #333333
- **RGB**: rgb(51, 51, 51)
- **Usage**: Body text, content

### Light Background
- **Hex**: #f5f5f5
- **RGB**: rgb(245, 245, 245)
- **Usage**: Section backgrounds, cards

### Border Color
- **Hex**: #dddddd
- **RGB**: rgb(221, 221, 221)
- **Usage**: Dividers, borders

### White
- **Hex**: #ffffff
- **RGB**: rgb(255, 255, 255)
- **Usage**: Main background, text on dark backgrounds

## Accessibility Notes

- Primary Blue (#2c5aa0) on White: Contrast ratio 8.5:1 ✓ WCAG AAA
- Secondary Blue (#6db3f2) on White: Contrast ratio 4.9:1 ✓ WCAG AA
- Accent Orange (#f39c12) on White: Contrast ratio 6.7:1 ✓ WCAG AAA
- Dark Text (#333) on Light Background: Contrast ratio 15.5:1 ✓ WCAG AAA

## Usage Guidelines

1. **Primary Blue** should be used for main interactive elements and branding
2. **Secondary Blue** should be used for hover states and secondary information
3. **Accent Orange** should draw attention to important actions and focus states
4. **Avoid** using only color to convey information; always include text labels
5. **Test** all color combinations for sufficient contrast ratios
