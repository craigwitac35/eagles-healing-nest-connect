# Eagles Healing Nest Connect

A community platform for residents of Eagles Healing Nest, featuring a clean, accessible, mobile-first design with organized content and clear navigation.

## Project Overview

Eagles Healing Nest Connect is a comprehensive website designed to serve the community with information, updates, events, resources, and volunteer opportunities.

## Features

- **Responsive Design**: Mobile-first approach for all devices
- **Accessible**: Built with accessibility standards in mind
- **Clear Navigation**: Intuitive site structure and user-friendly URLs
- **Community-Focused**: Sections for announcements, events, resources, and volunteer opportunities

## Pages

- **Home** (`/`) - Main landing page
- **About** (`/about/`) - Community information
- **Announcements** (`/announcements/`) - Latest news and updates
- **Events** (`/events/`) - Upcoming community events
- **Resources** (`/resources/`) - Helpful resources and information
- **Volunteer Opportunities** (`/volunteer/`) - How to get involved
- **Contact** (`/contact/`) - Get in touch with the community

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Navigate through the site using the menu

## Development

- Modify CSS in `css/styles.css`
- Add JavaScript functionality in `js/script.js`
- Add images to the `images/` folder organized by type
- Reference project documents in `docs/`

## Accessibility

This project follows WCAG 2.1 guidelines for web accessibility. All pages include:
- Semantic HTML structure
- ARIA labels where appropriate
- Color contrast compliance
- Keyboard navigation support
